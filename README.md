## 🚀 About Me

AI & Data Science undergraduate at **Jeppiaar Engineering College**, building at the intersection of **LLMs, AI agents, and workflow automation**.

- 🤖 Designing **AI-powered automation workflows** with **n8n** — connecting LLMs, APIs, and business tools into pipelines that run themselves
- 🦜 Building with **LangChain** — agents, RAG pipelines, and tool-calling LLM applications
- ⚡ Exploring how **agentic AI** can automate real work: data extraction, routing, summarization, and decision-making
- 📚 Strengthening my foundations in machine learning and deep learning along the way
